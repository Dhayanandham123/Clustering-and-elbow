### 1. K-Means Clustering
K-Means partitions the dataset into *K* clusters by minimizing the distance between data points and their respective cluster centroids. The algorithm iteratively updates centroids until convergence.

---

## ğŸ“ Elbow Method
The **Elbow Method** is used to determine the optimal number of clusters for K-Means.  
It plots the **Within-Cluster Sum of Squares (WCSS)** against different values of *K*.  
The point where the curve bends (elbow point) indicates the best number of clusters.
